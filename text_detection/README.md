# text_detection

<img src="../pic/text_detect.png" width=480/>

指定した画像からテキスト部分を検出し緑で囲んで描画。さらにそれら全部を含む領域をテキストの存在範囲として青で囲んで描画。
書類スキャン時のまわりの余分な部分をカットするなどに使えそう。
